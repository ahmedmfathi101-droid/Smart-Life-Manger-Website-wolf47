# Life OS - 11000 User Credentials

Only these usernames can sign in:

- `user001` through `user11000`
- Extra zero-padded variants like `user00001` also map to the same account number.

Each username has an independent profile, settings, entries, notes, goals, and dashboard data. Users are created lazily on first login so the browser database stays fast.
