# Life OS Dashboard

موقع عربي RTL لإدارة الحياة الشخصية من 9 أقسام:

1. الرئيسية
2. الصحة والجسم
3. العقل والتعلم
4. المال والعمل
5. العلاقات والمشاعر
6. المعنى والروح
7. الأولويات
8. التقارير والإحصائيات
9. الإعدادات

## التشغيل

افتح ملف `index.html` مباشرة في المتصفح.

الحساب الأول الجاهز:

- اسم المستخدم: `user001`
- كلمة المرور: موجودة في `CREDENTIALS.md`

الدخول محصور في 11000 مستخدم فقط من `user001` حتى `user11000`، وكل حساب له كلمة مرور مختلفة وبيانات مستقلة. يتم إنشاء ملف بيانات المستخدم عند أول دخول حتى يظل الموقع سريع. يمكن تغيير كلمة المرور من تبويب تغيير كلمة المرور أو من صفحة الإعدادات.

## التخزين

البيانات محفوظة محليا في المتصفح باستخدام `localStorage` كقاعدة بيانات مجانية بسيطة لكل مستخدم. يوجد أيضا ملف `firebase-config.js` لتفعيل مزامنة Firestore عند إنشاء مشروع Firebase لاحقا.

من صفحة الإعدادات يمكن:

- تعديل بيانات المستخدم وصورته.
- تعديل الأهداف اليومية.
- تصدير قاعدة البيانات بصيغة JSON.
- استيراد قاعدة بيانات سابقة.
- تصدير تقرير رسمي عبر نافذة PDF/Print.
- تصدير حسابات الدخول CSV.
- تصدير تقرير رسمي للطباعة PDF من نافذة الطباعة.
- تفعيل Gemini AI من `ai-config.js`.
- تفعيل أسعار العملات والذهب من `market-config.js`.

## Firebase

لتفعيل Firebase لاحقا:

1. أنشئ مشروع Firebase على خطة Spark المجانية.
2. فعّل Cloud Firestore.
3. انسخ بيانات Web app config داخل `firebase-config.js`.
4. غيّر `enabled` إلى `true`.
5. افتح الموقع من سيرفر محلي أو انشره على Firebase Hosting.

## الملفات

- `index.html`: هيكل الموقع والشاشات الأساسية.
- `styles.css`: التصميم المتجاوب المشابه للوحة المطلوبة.
- `app.js`: قاعدة البيانات المحلية، تسجيل الدخول، النماذج، الحسابات، التقارير، والربط بين الأقسام.
- `firebase-config.js`: إعدادات Firebase الاختيارية.
- `ai-config.js`: إعدادات Gemini AI الاختيارية.
- `market-config.js`: مصادر أسعار العملات والذهب.
- `CREDENTIALS.md`: نظام بيانات دخول الـ 11000 مستخدم وطريقة تصدير CSV كامل.
